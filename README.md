# 🚀 devChart – Student Club Collaboration Platform

## 📌 Project Overview

devChart is a Student Club Collaboration Platform designed to help student clubs efficiently manage tasks, announcements, team members, and club activities in one centralized workspace.

The platform provides role-based access, task tracking, announcements management, and a Kanban-style workflow to improve collaboration among club members and organizers.

### 🔐 Authentication & Authorization
- User Registration
- User Login
- Role-based Access Control (Admin/User)

## ⚙️ Setup Instructions

### 1. Clone Repository

```bash
git clone https://github.com/varunigabv2025/devchart.git
```

### 2. Navigate to Project

```bash
cd devchart
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Configure Environment Variables

Create a `.env.local` file and add:

```env
MONGODB_URI=your_mongodb_connection_string
```

### 5. Run Development Server

```bash
npm run dev
```

Application will run at:

```txt
http://localhost:3000
```

---

## 🚀 Deployment Instructions

### Deploy on Vercel

1. Push project to GitHub
2. Login to Vercel
3. Import GitHub Repository
4. Add Environment Variable:

```env
MONGODB_URI=your_mongodb_connection_string
```

5. Click Deploy

The application will be available through a Vercel deployment URL.
